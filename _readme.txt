Bienvenue dans l'application RSS Reader ! Guide d'utilisation.

- Menu : Vous avez la possibilité d'ajouter un flux en cliquant sur l'icône d'ajout du menu (il est nécessaire de renseigner un nom et un url valide de flux rss) et également de modifier les paramètres utilisateurs pour les options de mise à jour de l'application via l'option Settings du menu qui ouvre la page des préférences.

- Sur la page d'accueil : Gérez vos flux rss comme vous le souhaitez. Modifiez les flux que vous avez ajoutés via l'icône d'édition
(petit stylo). Pour supprimer un flux, cliquez longtemps sur ce dernier. Cette action ouvrira un message d'alerte pour valider ou non
la suppression du flux.

- Sur la page de détails d'un flux : Vous avez également la possibilité de rafraîchir la page en glissant vers le bas pour faire une mise à jour
manuelle.

Fonctionnalités supplémentaires :

- Selon la langue du téléphone, le contenu est soit en français soit en anglais.
- Modification et suppression des flux
- Récupération manuelle des éléments d'un flux grâce à un glisser vers le bas.
- Affichage du temps depuis la dernière synchronisation d'un flux.
- Possibilité de retour à l'activité parente via la flèche "précédent" du menu
- Les flux ne se mettent pas à jour automatiquement que si cela fait plus de 15 minutes que celui-ci a été mis à jour
