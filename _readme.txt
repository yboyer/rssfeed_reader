Bienvenue dans l'application RSS Reader ! Guide d'utilisation.

- Menu : Vous avez la possibilité d'ajouter un flux en cliquant sur l'icone d'ajout du menu (il est nécessaire de renseigner un nom et un url valide de flux rss) et également de modifier les paramètres utilisateurs pour les options de mise à jour de l'application via l'option Settings du menu qui ouvre la page des préférences.

- Sur la page d'accueil : Gérez vos flux rss comme vous le souhaitez. Modifiez les flux que vous avez ajoutés via l'icone d'édition
(petit stylo). Pour supprimer un flux, cliquez longtemps sur ce dernier. Cette action ouvrira un message d'alerte pour valider ou non
la suppression du flux.

- Sur la page de détails d'un flux : Affiche les détails du flux rss que vous avez ajouté précédemment. On peut visionner un titre, une url ainsi qu'une description. Lors du clic sur l'url, l'application ouvre le navigateur web du téléphone pour se rendre à
l'adresse en question. Vous avez également la possibilité de rafraichir la page en glissant vers le bas pour faire une mise à jour
manuelle.

Fonctionnalités supplémentaires :

- Selon la langue du téléphone, le contenu est soit en français soit en anglais.
- Mise à jour manuelle des détails d'un flux grâce à un glisser vers le bas.
- Affichage du temps passé depuis la dernière mise à jour d'un flux en dessous du titre d'un flux sur la page de détails.
- Pas de soucis d'affichage que ce soit en mode portrait ou en mode paysage.
- Tri alphabétique des flux
- Possibilité de retour à l'activité parente via la flèche "précédent" du menu
- Gestion des erreurs lors de l'insertion d'un flux
